# Portafolio Personal

Sitio personal construido con Astro, Tailwind CSS y MDX.

## Stack Tecnológico

- **Framework:** Astro v5
- **Estilos:** Tailwind CSS
- **Tipografía:** Inter (Google Fonts)
- **Contenido:** MDX con Content Collections
- **Hosting:** Vercel

## Colores del Sistema

- **Fondo:** `#FAF9F6` (blanco cálido)
- **Texto:** `#1A1A1A` (negro fuerte)
- **Acento:** `#0066FF` (azul cobalto)

## Instalación

```bash
# Instalar dependencias
npm install

# Iniciar servidor de desarrollo
npm run dev

# Construir para producción
npm run build

# Previsualizar build de producción
npm run preview
```

## Estructura del Proyecto

```
src/
├── layouts/          # Layouts base del sitio
├── components/       # Componentes reutilizables
├── pages/           # Páginas del sitio
├── content/         # Contenido MDX (writing + work)
└── styles/          # Estilos globales
```
